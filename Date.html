Subject: Automatically Updating Date/Time in Word Documents

Purpose

This memorandum provides instructions on how to insert an automatically updating date and time in Microsoft Word documents.

Step 1: Insert Auto Date/Time Using the Ribbon

  • Place your cursor where you want the date and time to appear.
  • Click Insert > Date & Time.
  • Choose your preferred date and time format.
  • Check Update automatically to ensure it updates each time the document is opened.
  • Click OK.

Step 2: Insert Auto Date/Time Using a Field Code

  • Place your cursor where you want the date/time to appear.
  • Press Ctrl + F9 to insert field brackets { }.
  • Type DATE (for the current date) or TIME (for the current time).
  • Press F9 to update the field.
{ DATE } or { TIME }

Step 3: Lock or Unlock Auto Date/Time

  • To **lock** the date/time (prevent auto-updates), select the field and press Ctrl + F11.
  • To **unlock** it, select the field and press Ctrl + Shift + F11.

Conclusion

By following these steps, you can ensure that your Word documents always reflect the current date and time without manual updates.
