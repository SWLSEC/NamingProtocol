Memorandum: Restricting Word Documents with Fillable Fields

To: Legal Clinic Staff

From: Nolan Silverstein

Date: 2/24/2025

Subject: Protecting Word Documents While Allowing Client Input

Purpose

This memorandum provides instructions on how to restrict editing in Microsoft Word while allowing clients to fill out designated form fields.

Step 1: Enable the Developer Tab

  • Open Microsoft Word.
  • Click File > Options.
  • Select Customize Ribbon.
  • Check the box for Developer and click OK.

Step 2: Insert Fillable Form Fields

  • Click the Developer tab.
  • Select Design Mode.
  • Choose either Rich Text Content Control (allows formatting) or Plain Text Content Control (text-only).
  • Click where you want the field and insert the control.

Step 3: Restrict Editing

  • Click Developer > Restrict Editing.
  • Under Editing Restrictions, check Allow only this type of editing and select Filling in forms.
  • Click Yes, Start Enforcing Protection and enter a password (optional).

Step 4: Save and Distribute

  • Click File > Save As.
  • Choose .docx format.
  • Distribute the document to clients for completion.

Conclusion

By following these steps, we can ensure that legal documents remain protected while allowing clients to input necessary information.
